Hexo博客搭建感想

Created2020-05-01|Updated2020-05-11
|Post View:

liumou的Hexo博客搭建感想

算法考题的时候，本着诚实守信的精神做题，想着2个题是做过的原题，我当时心理狂喜，却没想到倒在了两个做过的原题上，一道题脑袋进水错误，另一个看着*打成了X（太受原题影响了），直接0分。

怀着悔恨开始了占比更大的项目，只有通过努力来弥补自己犯得错误。

总体来说，做博客困难不是特别大，并且流程也比较简单，但是其中遇到的bug和一些问题真的让这几天的我，吃不饱，睡不香。下面来说说我碰到的几个bug。

1.本地图片无法上传

一天没有课，本打算在4.29号就把blog搭建完成，可是设置头像时本地图片无法上传，在百度上只找到了两种方案，这两种方案都不成功。也麻烦了很多学长学姐（后端学长都来看了），都不知道怎么回事，研究了一整天之后，发现目前只能暂时在config文件下上传图床处理过的图片。=-=

解决方法:上传经过图床处理过的图片

2.hexo d->Please make sure you have the correct access rights and the repository exists

在经过大量的资料查证并且实地考察后，发现问题所在——git服务器没有存储本地ssh密钥，在不断产生过后终于成功了！

解决步骤：

1.删除 .ssh 文件夹【C:\Users(本地用户名).ssh】 中的 known_hosts(直接删除即可)

2.在下载好的Git中的bin目录下（一般是 C:\Program Files\Git\bin）打开bash.exe输入命令ssh-keygen -t rsa -C “username” (注：username为你git上的用户名)，如果执行成功。返回：

       Generating public/private rsa key pair.

       Enter file in which to save the key (/Users/username/.ssh/id_rsa):
//这里的username是电脑上的用户名，这个地址也是文件的存储地址，然后按回车，

如果以前有存储地址会返回/Users/your username/.ssh/id_rsa already exists.Overwrite (y/n)?直接输入y回车。

如果以前没有储存地址就会出现Enter passphrase(empty for no passphrase);也直接回车，

两种情况回车后都会出现 Enter same passphrase again 然后接着回车会显示

The key’s randomart image is:（大概）

+—[RSA 2048]—-+

|    .          . |

| . = .       .  o|

|o o * .     . …|

|E oo o   .   o.. |

| B .  o S . …  |

|. o    o .o..    |

| .       o**.    |

|        .B=+%.   |

|         +*BoBo  |

+—-[SHA256]—–+

这说明SSH key就已经生成了。文件目录就是：username/.ssh/id_rsa.pub.

3.然后找到系统自动在.ssh文件夹下生成两个文件，id_rsa和id_rsa.pub，用记事本打开id_rsa.pub将全部的内容复制。

4.打开https://github.com/，登陆账户，进入设置（Settings）找到

5.然后将你复制的内容粘贴到key中,再点击Add SSH Key  

6.仍然在bash.exe中输入ssh -T git@github.com然后会跳出一堆内容你只需输入yes回车就完事了，然后他会提示你成功了。

大功告成，再次输入 git clone 就成功了

参考方法：前辈给出了相关方案

3.看板娘手机显示不了

解决方法：yilia/config文件中mobile show:true

一个人可能走的更快，但是一个团队会走的更远（一点心里话）

在做完所有项目之后，还有几个小时就要提交项目了，心理还是有点紧张吧。之前是嵌入式团队成员，后来不知道为什么就没参加了，现在想起来怎么也是后悔莫及，还是很想再回到一个团队里面，去和团队的成员做一些项目，学很多东西。

其实一个团队会走的更快，并且更远。

因为这件事本来就需要很大的毅力与恒心，退出团队的时候，我天天和室友再寝室里面玩游戏，该上课就去，期末复习一下，绩点和成绩还勉强过的去，但是回想起来，感觉大学里面什么都没做，和自己想的差距挺远的。以前在团队的时候，还每天去打卡，争取多学一点知识，虽然有时挺累，但是感觉收获挺大的。

现在也想通了，希望自己能够学有所成，能认识更多志同道合的人，以前把流浪当成梦想，生活没有辜负我，但给我的是心在流浪。等待不会等来你想要的，要出发才有收获。努力了，更会有意想不到的惊喜。梦想，你相信始终会到来，那就是信念。
希望自己这次的项目能够让学长学姐满意。这次的项目我自己还是比较满意的。

祝你我年少有为！

Wish we young and promising！
